# EvalForge 🛠️

**The Open Standard for Agent Evaluation & Benchmarking**

EvalForge is a comprehensive platform for rigorously evaluating AI agents. It provides visual replay, deep diagnostics, reliability scoring, and actionable insights.

## Features

- **Visual Replay** — Step through any agent session to see exactly what happened
- **Auto Diagnostics** — Root-cause analysis for failures (looping, context exhaustion, tool errors)
- **Cost Tracking** — Token and cost analysis per run
- **Comparison Views** — Side-by-side metrics across runs
- **Standard Benchmarks** — Coding, research, and tool-use evaluation tasks
- **Provider Agnostic** — Works with any LLM (DeepSeek, Gemini, Groq, Ollama, etc.)

## Usage

Visit the dashboard to create evaluations, run benchmarks, and analyze results.
